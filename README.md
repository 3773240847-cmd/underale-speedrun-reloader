# underale-speedrun-reloader
a faster restart python for undertale speedruns


f1 restart the game(f1直接重启游戏)


f2 restart the game with cleaning all files(f2清存档重启游戏)


f6 restart the game with Start at Punchcard file(f6用sapc存档重开游戏)

f6 only run successfully with a %Appdata%\Undertale_linux_steamver\SAPC.zip
